#####################################################################
Легенда:
	( ) - невыполненная задача, работа в текущей ветке не начиналась
	(*) - невыполненная задача, работа в текущей ветке началась
	(v) -   выполненная задача, работа в текущей ветке закончилась

Пример:
Допустим, работа ведётся над задачей task_main, состоящей из подзадач task_a, task_b, task_c, task_d. В ветке branch_a ведётся работа над task_a, в ветке branch_b закончилась работа над task_c и ведётся работа над task_d. Файл plan.txt в этих ветках:

branch_a: plan.txt
1. (*) task_main:
	1.1 (*) task_a
	1.2 ( ) task_b:
		- ( ) task_c
		- ( ) task_d

branch_b: plan.txt
1. (*) task_main:
	1.1 ( ) task_a
	1.2 (*) task_b:
		- (v) task_c
		- (*) task_d
